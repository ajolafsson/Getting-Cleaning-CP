    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="user-content-getting-and-cleaning-data-course-project" class="anchor" href="#getting-and-cleaning-data-course-project" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting and Cleaning Data Course Project</h1>

<p>This is the repo for the final project for Getting and Cleaning Data Course.</p>

<p>This repo contains the following files and folders. </p>

<ul class="task-list">
<li>
<strong>run_analysis.R</strong> The r script processes the data and generated the required tidy dataset for submission. </li>
<li>
